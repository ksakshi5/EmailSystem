# 💌 Mail System (Console-Based Java Project)

A simple console-based email management system built using Java.  
This project simulates basic functionalities of an email service like Gmail, including account creation, login, composing mails, sending, drafts, and inbox management.

---

## 🚀 Features

- 👤 User Registration & Login
- 📧 Compose Mail
- 📤 Send Mail
- 📥 Inbox Management
- 📝 Draft Mail Editing & Sending
- 🔍 Email Validation System
- 💡 Username Suggestions
- 🖥️ Console-Based Interactive UI

---

## 🛠️ Technologies Used

- Java
- OOP Concepts (Encapsulation, Abstraction)
- ArrayList (Dynamic Data Storage)
- Java Time API (`LocalDate`, `LocalTime`)
- Scanner (User Input Handling)

---

## 📂 Project Structure
- Mail Class -> Handles email structure (date, time, sender, receiver, subject, description)
- User Class -> Stores user details and mail folders (Inbox, Sent, Draft)
- Gmail Class -> Core logic (login, compose, send, draft handling)
- DriverMail -> Main class to run the application
